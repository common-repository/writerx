WriterX integrates your WordPress site with the WriterX.ai service, enabling automatic content generation and publication at scale.

== Description ==
WriterX is a powerful plugin designed to integrate your WordPress site seamlessly with the WriterX.ai service. With this plugin, you can effortlessly connect to your WriterX.ai account via API, enabling the automatic generation and publication of high-quality content at scale. Whether you're managing a small blog or a large website, WriterX allows you to automate your content creation process, saving you time and resources.

It allows you to retrieve comprehensive data from your website, ensuring that you have all the information you need for effective content management and strategy.

Perfect for marketers, content creators, and businesses looking to streamline their content production, WriterX offers a reliable and scalable solution to keep your website updated with minimal effort.

== Installation ==
1. Upload the `writerx` folder to the `/wp-content/plugins/` directory or install the plugin directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Navigate to the 'WriterX' settings page.
4. Copy the API key displayed on the plugin page.
5. In your WriterX.ai account, add a new WordPress site. 
6. When prompted, paste the API key on site settings screen in your WriterX dashboard.
